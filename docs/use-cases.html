In Practice

Here's what it actually looks like to integrate x402 Charity into a real product.

DeFi Example

A DEX with 50K daily trades

Add one middleware to your swap endpoint. Every POST request fires a $0.001 USDC donation. Non-blocking — your trading latency stays the same.

"Every swap you make helps fund clean water."

POST /api/swap → $0.001 USDC donated 
import express from 'express';
import { x402charity } from '@x402charity/express';

const app = express();

app.use('/api/swap', x402charity({
  privateKey: process.env.KEY,
  cause: 'clean-water',
  amount: '0.001',
  shouldDonate: (req) =>
    req.method === 'POST',
}));
50K trades / day
$50 donated / day
$18.2K / year auto
AI Product Example

An AI API with 100K daily calls

Attach to your chat endpoint in Next.js. Every inference request donates to STEM education. Show users a live impact counter on your dashboard.

"Every AI response contributes to a better world."

/api/chat/* → $0.0001 USDC donated 
// middleware.ts
import { x402charity } from '@x402charity/next';

export default x402charity({
  privateKey: process.env.X402_PRIVATE_KEY,
  cause: 'stem-education',
  amount: '0.0001',
  matcher: '/api/chat/*',
});
