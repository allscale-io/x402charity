Programmable Giving

Attach a cause to any event — a trade, an API call, a subscription. Donations happen in the background.

Powered by x402

Uses the x402 protocol for native HTTP stablecoin payments. If payments can be programmable, so can giving.

Open Source

MIT licensed. Add your charity to the registry. Integrate with any x402-enabled app.

Add donations in 4 lines

const charity = x402Charity({
  cause: 'red-cross',
  amount: '0.0001',  // USDC per event
});

app.post('/trade', charity.wrap(async (req, res) => {
  // your trade logic — donation happens automatically
}));
